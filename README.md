# 🎮 Guess the Next Display – VB.NET Game

**Guess the Next Display** is a simple video game built using **VB.NET**.  
The objective of the game is to predict or guess the **next item that will be displayed** on the screen. Players make a guess, and the game determines whether the prediction is correct.

This project was developed as a **learning and demonstration project** to illustrate how to build a basic game using **VB.NET and the .NET framework**.

---

# 🚀 Project Overview

The game presents a sequence of displays (such as numbers, symbols, or images), and the player must **guess what will appear next**. If the guess is correct, the player earns points; otherwise, the game continues to the next round.

The application demonstrates:

- Basic **game logic**
- **Event handling**
- **User interface interaction**
- Simple **game state management**

---

# 🎯 Game Objective

The goal of the game is simple:

1. Observe the current display.
2. Guess what the **next display** will be.
3. Submit your guess.
4. The game checks whether your guess is correct.

Points are awarded for correct guesses.
